# Численные методы решения дифференциальных уравнений

Этот репозиторий содержит реализацию лабораторной работы по численному решению дифференциальных уравнений с использованием методов Эйлера и Хьюна. Программа визуализирует результаты расчётов, позволяет настроить параметры и исследовать влияние случайного шума.

## Функциональность
- Реализация методов:
  - **Метод Эйлера**
  - **Метод Хьюна**
- Визуализация результатов:
  - Графики зависимости `x(t)` и `dx/dt` (с шумом и без).
- Поддержка пользовательских настроек:
  - Параметр уравнения `a`.
  - Максимальное время моделирования.
- Управление параметрами через удобный GUI.

## Используемые технологии
- **C++** — основной язык разработки.
- **Qt** — для построения интерфейса и графиков.
- **QtCharts** — для визуализации данных.

## Структура проекта
- `MainWindow.cpp` — главный файл приложения, содержит настройку вкладок и основного интерфейса.
- `EulerWidget.cpp` — отвечает за построение графиков для метода Эйлера.
- `EulerModel.cpp` — содержит реализацию численного метода Эйлера, включая добавление случайного шума.
- `HeunWidget.cpp` — отвечает за построение графиков для метода Хьюна.
- `HeunModel.cpp` — содержит реализацию численного метода Хьюна, включая добавление случайного шума.
